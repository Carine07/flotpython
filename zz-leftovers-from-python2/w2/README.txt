Il y a dans ce répertoires tout le matériel pour faire le MOOC sur une
semaine.

Il y a trois types de fichiers:

-CO12AL-Wi-PROMPTj.txt est le fichier numéro j contenant le texte à
dire face caméra pour la semaine i.

-CO12AL-Wi-SLIDEj.txt est le fichier numéro j contenant un slide à
 montrer pour la semaine i.

-CO12AL-Wi-EXOj.py est le fichier numéro j contenant le code à taper
 pendant que je parle pour illustrer l'utilisation de Python pour la
 semaine i.

Pour éviter la renumérotation des fichiers en cas de modification, le
fichier COAL12-Wi-SUM.txt contient l'ordre de l'enchaînement des
fichier PROMPT, SLIDE, EXO pour chaque séquence dans la semaine.
